## Project Structure

Each project should have:
- `README.md` - Project overview with frontmatter
- `Docs/` - Documentation (optional)
- Project-specific folders as needed

### Project README Template

```yaml
---
type: project
status: active
lead: "[[Person Name]]"
started: YYYY-MM-DD
tags:

---
```
